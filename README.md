# 3m 안전 모니터링 시스템

ESP32 UWB DW3000을 활용한 건설 현장 작업자 안전 모니터링 시스템입니다.

## 프로젝트 소개

건설 현장에서 작업자와 중장비 간의 거리를 실시간으로 측정하여 위험 상황을 감지하고 경고를 제공하는 시스템입니다.

UWB(Ultra-Wideband) 기술을 이용하여 작업자가 위험 거리(3m) 이내로 접근할 경우 즉시 경고 신호를 발생시켜 사고를 예방하는 것을 목표로 합니다.

## 주요 기능

### 현재 구현 완료

* UWB 기반 거리 측정
* Anchor ↔ Tag 실시간 통신
* 위험 거리 판단 로직
* GPIO 경고 신호 출력
* 거리 데이터 실시간 모니터링
* 히스테리시스(Hysteresis) 적용

### 개발 예정

* 다중 Tag 지원
* Wi-Fi 연결
* MQTT 통신
* GPS 위치 정보 연동
* 실시간 웹 대시보드
* 지도 기반 모니터링
* 이벤트 로그 저장
* 이메일/SMS 알림

## 시스템 구성

### Anchor (중장비)

* ESP32 UWB DW3000
* GPS 모듈 (예정)
* GPIO 경고 출력

### Tag (작업자)

* ESP32 UWB DW3000

## 동작 방식

1. Anchor와 Tag가 UWB를 이용하여 거리 측정
2. 작업자가 위험 거리(3m 이하) 진입
3. Anchor가 즉시 위험 상태 판단
4. GPIO를 통해 경고 신호 출력
5. 작업자가 안전 거리로 이동 시 경고 해제

## 안전 로직

### 위험 상태

* 거리 ≤ 3.0m
* 즉시 경고 활성화

### 안전 상태 복귀

* 거리 ≥ 3.2m
* 연속 확인 후 경고 해제

거리 경계 부근에서 경고가 반복적으로 켜지고 꺼지는 현상을 방지하기 위해 히스테리시스 로직을 적용하였습니다.

## 시스템 아키텍처

```text
작업자(Tag)
      │
      │ UWB
      ▼
장비(Anchor)
      │
      │ Wi-Fi / MQTT
      ▼
MQTT Broker
      │
      ▼
Node.js 서버
      │
      ▼
웹 대시보드
```
